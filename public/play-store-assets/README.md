## Required Assets

### 1. App Icon ✅
- **Size:** 512 x 512 px
- **Status:** Available at `../icons/icon-512x512.png`
- **Action:** Copy to `icon/` folder when ready to submit

### 2. Feature Graphic ⚠️
- **Size:** 1024 x 500 px
- **Format:** PNG or JPEG
- **Location:** `feature-graphic/feature-graphic-1024x500.png`
- **Status:** NEEDS TO BE CREATED
- **See:** `../../../PLAY_STORE_ASSETS.md` for design guidelines

### 3. Phone Screenshots ⚠️
- **Size:** 1080 x 1920 px (portrait) recommended
- **Minimum:** 2 screenshots
- **Recommended:** 4-8 screenshots
- **Location:** `screenshots/phone/`
- **Status:** NEEDS TO BE CREATED
- **See:** `../../../PLAY_STORE_ASSETS.md` for capture instructions

## Quick Reference

| Asset | Size | Min | Max | Format |
|-------|------|-----|-----|--------|
| App Icon | 512x512 | 1 | 1 | PNG (32-bit) |
| Feature Graphic | 1024x500 | 1 | 1 | PNG or JPEG |
| Phone Screenshots | 1080x1920 | 2 | 8 | PNG or JPEG |

### Design Guidelines
- **Brand Color:** #3F9D52 (Green)
- **App Name:** Tailors Suite
- **Style:** Professional, clean, modern
- **Content:** Show real app screens, not mockups

## Creating Assets

### For Feature Graphic:
1. Use design tool (Canva, Figma, Photoshop)
2. Create 1024x500 canvas
3. Add app name and key visual
4. Use brand colors
5. Export as PNG
6. Save to `feature-graphic/`

### For Screenshots:
1. Run app on Android emulator (1080x1920)
2. Navigate to key screens
3. Take screenshots
4. Optional: Add device frames using mockuphone.com
5. Save to `screenshots/phone/`

## Tools

- **Canva:** Easy templates - https://canva.com
- **Figma:** Professional design - https://figma.com
- **Screenshots.pro:** Add device frames - https://screenshots.pro
- **Mockuphone:** Device mockups - https://mockuphone.com
